The IfcPolyline is a bounded curve with only linear segments defined by a list of Cartesian points. If the first and the last Cartesian point in the list are identical, then the polyline is a closed curve, otherwise it is an open curve.

EXAMPLE  Figure 2 illustrates a bounded IfcPolyline and shows the parametric length of each segment and of the total polyline. The parametric length of the entire polyline is n - 1, where n is the number of Points.

NOTE  Definition according to ISO/CD 10303-42:1992
A polyline is a bounded curve of n - 1 linear segments, defined by a list of n points, P1, P2 ... Pn. The i th segment of the curve is parameterized as follows:
    for 1 ≤ i ≤ n - 1
where i - 1 ≤ u ≤ i and with parametric range of 0 ≤ u ≤ n - 1.
