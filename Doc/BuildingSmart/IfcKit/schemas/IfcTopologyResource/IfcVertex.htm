NOTE  Definition according to ISO/CD 10303-42:1992
A vertex is the topological construct corresponding to a point. It has dimensionality 0 and extent 0. The domain of a vertex, if present, is a point in m dimensional real space RM; this is represented by the vertex point subtype.
NOTE  Entity adapted from vertex defined in ISO 10303-42.
HISTORY  New entity in IFC2.0

Informal Propositions:

  1. The vertex has dimensionality 0. This is a fundamental property of the vertex.
  2. The extent of a vertex is defined to be zero.
